# AI Agent Definition Template

Use this template to define AI agents following SOLID.AI principles.

For the complete YAML template, see:

**[→ Full Template on GitHub](https://github.com/gusafr/midora-solid-ai/blob/main/ADOPTION/TEMPLATES/agent-definition-template.yaml)**

---

## Template Sections

The template includes:

1. **Agent Identity** - Name, role, persona
2. **Purpose Statement** - Why it exists, mission alignment
3. **Capabilities** - What it CAN do
4. **Guardrails** - What it CANNOT do
5. **Human Oversight** - When humans intervene
6. **Success Metrics** - Value, performance, ethics
7. **Observability** - Logging, dashboards, alerts
8. **Learning & Iteration** - How it improves
9. **Data & Privacy** - What data, how protected
10. **Failure & Rollback** - Safety mechanisms

---

## Usage Instructions

1. **Copy the template** to your repository
2. **Fill in each section** - use the [AI Agent Definition Prompt](../prompt-templates/ai-agent-definition.md)
3. **Review with team** - engineers, product, ethics reviewers
4. **Publish to agent registry**
5. **Update as agent evolves**
