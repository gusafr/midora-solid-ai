# ADR Template

Use this template to document architectural decisions.

For the complete markdown template, see:

**[→ Full Template on GitHub](https://github.com/gusafr/midora-solid-ai/blob/main/ADOPTION/TEMPLATES/adr-template.md)**

---

## Template Sections

The template includes:

1. **Context** - Issue and driving forces
2. **Decision** - What we're choosing
3. **Rationale** - Why we're making this decision
4. **Alternatives Considered** - Other approaches evaluated
5. **Consequences** - Positive, negative, neutral outcomes
6. **Implementation Notes** - Practical guidance
7. **Related Decisions** - Links to other ADRs/RFCs

---

## Usage Instructions

1. **Copy the template** to your repository
2. **Fill in each section** concisely
3. **Review with team** for technical soundness
4. **Publish** to your ADR folder
5. **Reference** in related code/docs
6. **Update status** if superseded
