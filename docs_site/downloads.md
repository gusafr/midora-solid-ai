##  Which PDF Should I Download?

**Choose the Whitepaper if you:**
- Need to cite SOLID.AI in academic or professional work
- Want a concise overview of the framework (~50 pages)
- Prefer academic typography and formatting
- Are presenting to executives or stakeholders
- Need a publication-ready document

**Choose the Complete Framework if you:**
- Are implementing SOLID.AI in your organization
- Need detailed playbooks and adoption guides
- Want reference cards for specific roles or industries
- Are conducting training sessions
- Need prompt templates and checklists
- Want the full technical reference
